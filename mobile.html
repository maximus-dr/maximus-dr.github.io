Обо мне

С 2018 года все свободное время и ресурсы вкладываю в изучение фронтенд разработки. Моя цель – построение карьеры в этом направлении. Отправной точкой моего обучения стали три интенсивных курса HTML Academy, посвященных разработке и верстке.

Также в качестве источника знаний я использую онлайн-учебники, вебинары, лекции, статьи, интерактивные уроки. Слежу за тенденциями развития индустрии в целом и фронтенд-разработки в частности.

В данный момент у меня есть опыт создания SPA приложений, опыт верстки адаптивных, отзывчивых интерфейсов. В ближайших планах – освоить TypeScript, Jest, работу с базами данных, язык серверного программирования.

Есть большое желание применять накопленные знания и навыки в работе над коммерческими проектами, развиваться профессионально, работать в команде, осваивать новые технологии и практики разработки.

Мой опыт

  • Сервис "Kekstagram"

    HTML Academy 2018 Санкт-Петербург
    Репозиторий

    Кекстаграм — сервис просмотра изображений. Пользователям предоставляется возможность загружать свои фотографии, применять к ним фильтры, просматривать и фильтровать фотографии, загруженные ранее другими пользователями. Весь функционал сервиса был написан мной в рамках курса Professional JavaScript level 1 - HTML Academy.

  • Онлайн-игра "Pixel-hunter"

    HTML Academy 2018 Санкт-Петербург
    Репозиторий

    Пиксель Хантер — игра, в которой игроку предлагается отличать фотографии от фотореалистичных изображений. Работа над проектом проходила в рамках продвинутого интенсива Professional JavaScript level 2 - HTML Academy. Моей задачей было проектирование архитектуры и логики приложения, написание всего функционала и тестов.

  • Лендинг магазина игрушек "Mishka"

    HTML Academy 2018 Санкт-Петербург
    Репозиторий

    Мишка - онлайн-магазин изделий ручной работы. Сайт содержит множество нестандартных компонентов интерфейса, декоративных элементов, перестраивающуюся сетку. Это личный проект на продвинутом интенсиве Professional HTML & CSS level 2 - HTML Academy, в рамках которого необходимо было полностью сверстать сайт, настроить автоматизацию, оптимизировать изображения, шрифты, загрузку.
